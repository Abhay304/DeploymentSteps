Configure MongoDb

  1. Go To https://www.mongodb.com/cloud/atlas
  2. Create Your ID.
  3. Create your Free Account.
  4. After creating Free Acount, Your Cluster 0 will be created
  5. Now, Go to Database acess and create New User By Clicking on ADD NEW USER DATABASE.
  6. Generate your password and copy that password for connecting your application with MongoDB.
  7. Now, Go to Network acess and WhiteList Your IP by using ADD Current IP address or Copy paste the IP you want to Whitelist.
  8. Now Go to Clusters and click on connect.
  9. Now Select Connect your application.
  10. Copy the connection string and paste it into your application code and change username password and dbname.
    mongodb+srv://'username''password'@cluster0-juap7.mongodb.net/'dbname'?retryWrites=true&w=majority
  11. Now click on collections and then Create DataBase(This is your dbname) and then create your schema also.
  12. Now Your MongoDb is configured.
  13. Now Just connect mongoDb with your application.

Deploy on Git Hub

Using Vs Code for deployment

Normal Appdeployment on GitHub Pages

  1. Save your all files in one folder and most important main html as index.html
  2. Create a repository on Git Hub
  3. Create your Readme File on github
  4. Upload All your Files on Git Hub
  5. Go To the settings of the repository you want to publish
  6. Got to GitHub Pages and select Master branch in dropdown
  7. Wait for Some minute and it will reflect

Angular App deployment on GitHub Pages

  1. git init
  2. git add README.md
  3. git commit -m "first commit"
  4. git remote add origin https://github.com/Abhay304/demo.git ("projectName")
  5. git push -u origin master
  6. npm i -g angular-cli-ghpages --save
  7. ng build --prod --base-href "https://username.github.io/repository_name/"
    Example ng build --prod --base-href "https://Abhay304.github.io/Angular_deploy_demo/"
  8. ngh --dir=dist/[project-name] Or angular-cli-ghpages
  9. Check the settings and under that Git Hub Pages

Deploy your Frontend on Amazon S3

Using Vs Code for deployment



If you are using angular project then follow the below mentioned steps after building your project for production-
Process to build for production are-
  1. In your Vs Code node console and under your angular folder type : ng build --prod
  2. Now after sucessfull build Dist folder will be created and you need to upload all the files generated (Mentioned below where to upload)

Normal App deployment on GitHub Pages

  1. Go to - https://aws.amazon.com/
  2. Create Account in AWS and search For S3.
  3. Create Bucket
  4. In DNS name - Enter Your Project Name
  5. After Creating Bucket
  6. Go To Public access and the turn off your block all public access and save It
  7. Go To Bucket Policy and then copy and Paste Below Mentioned Code and replace yourBucketName word to your Bucket Name and Save It
    				{
					"Version": "2012-10-17",
					"Statement": [
					{
						"Sid": "PublicRead",
						"Effect": "Allow",
						"Principal": "*",
						"Action": "s3:GetObject",
						"Resource": "arn:aws:s3:::yourBucketName/*"
					}
					]
				}
  8. Now Go to Properties
  9. Select Satic website hosting
  10. Give the Name of Index Document and Error Document.
    Example index.html for both the filed if you have not created Error.html
    Error.html is not mandatory You can use index.html for both.
  11. Now Save Without selcting any radio button or Redirection rules.
  12. Now Re open the Satic website hosting and you will get a link on the head of the card of Satic website hosting
  13. Bingo!! you have sucessfully hoisted you website.

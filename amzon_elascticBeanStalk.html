Deploy your server file on Amazon Elastic Bean Stalk

Using Vs Code for deployment

Deployment Server Files On Amazon S3

  1. Go to - https://aws.amazon.com/
  2. Create account in AWS and search for AWS Elastic Beanstalk.
  3. Click on Create Application.
  4. Give Your Application name and select your Platform
    I have selected Node Js as I mostly use Node.js for my projects. 
    {
	"name": "website",
	"version": "1.0.0",
	"private": true,
	"dependencies": {
	"body-parser": "^1.19.0",
	"express": "^4.17.1",
	"mongoose": "^5.9.12"
	}
}
  5. Click on upload your Code.
  6. Uploading Steps are mentioned below.
  7. Click on your Local file.
  8. Now Create Application.
  9. Now re open the Satic website hosting and you will get a link on the head of the card of static website hosting
  10. Bingo!! you have sucessfully hoisted you server.

Uploading Steps

  1. Go to your coding files folder where you have server.js and app.js files.
  2. Create a package.json file
  3. Create a Procfile
    a. Procfile content - web: node server.js
    b. Save the file with name procfile without any extension
  4. Now make a Zip file with the name Archive and .zip extension.
  5. Upload Your Archive.zip into the AWS Elastic Beanstalk.
